## 🚴 Usage

### 🛠️ Just run  

```
denon start
```

### 🔬 Result

  <strong>See
  <a href="http://localhost:8000/">http://localhost:8000/</a></strong>

### Docker

#### Build
```
docker-compose build
```

#### Run
```
docker-compose up deno
```

#### Result
  <strong>See
  <a href="http://localhost:5000/">http://localhost:5000/</a></strong>
